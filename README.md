# ScaleSoftPitzer (SSP2022)
ScaleSoftPitzer (SSP2022) 是一款在油气、化工和水处理行业中广泛使用的专业软件，主要用于预测水中矿物质结垢的趋势。

## 核心功能
ScaleSoftPitzer 是一个基于物理化学模型的计算机程序，用于计算和预测水溶液（特别是高含盐量的水，如海水、油气田采出水）中各种矿物质是否会在特定温度、压力条件下析出并形成水垢。

ScaleSoft： 意为“结垢软件”。

Pitzer： 指的是 Pitzer 电解质溶液理论。这是该软件的核心算法基础，由物理化学家 Kenneth Pitzer 开发。与简单的理论相比，Pitzer 理论能更精确地描述高离子强度溶液（如浓盐水）的热力学性质。

## 软件的详细功能描述
在许多工业过程中，水是必不可少的。当水的条件（如温度、压力、浓度）发生变化时，溶解在水中的矿物质会达到过饱和状态并析出，形成坚硬的固体沉积物，这就是“结垢”。

这些水垢会带来严重问题：

1. 堵塞管道和设备，降低生产效率。

2. 降低热交换器的传热效率，增加能耗。

3. 导致设备腐蚀，缩短设备寿命，增加维护成本和安全隐患。

ScaleSoftPitzer 的作用就是通过模拟计算，提前预测在给定的水质和工艺条件下，是否会产生水垢、会产生哪种水垢、产生的量有多少，以及需要在什么条件下（如添加阻垢剂）才能防止结垢。

## 软件开发
NewSSP软件：最新更新代码在`hzy/All_part.cpp`中

开发流程：

使用Excel内置的编译器调试VB代码，之后一步步对照着完善C代码，逐步完成Excel VBA代码转换为C语言的任务。

调试准备：

1、搭建VBA调试环境：确保Excel可以运行和调试宏（**启用开发工具**，设置宏安全级别）。

2、使用VBA调试器：设置断点，使用 F8（逐语句） 单步执行，在本地窗口观察所有变量的值，记录下输入参数和准确的输出结果。

3、使用C调试器，如GDB，将C调试器中每一步的变量值（尤其是循环计数器、条件判断分支、中间计算结果）与之前在VBA调试器中记录的值进行逐行、逐变量对比，**使用 `printf` 大法**。
